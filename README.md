## Setup
To run this project, install it locally using [Yarn](https://classic.yarnpkg.com/en/docs/install/#debian-stable):

```
$ git clone https://github.com/mohamedMghazi/eCommerce-core.git
$ cd eCommerce-core\
$ yarn
$ yarn start
```
